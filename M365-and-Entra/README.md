# Microsoft 365 Administration Projects

This section contains practical Microsoft 365 administration projects focused on identity, messaging, collaboration, and cloud file management.

## Projects

### Entra Tenant Management
Hands-on administration of Microsoft Entra ID, including users, groups, licensing, and identity management.

### Exchange Online
Exchange Online projects covering shared mailboxes, mailbox delegation, distribution groups, aliases, and message tracing.

### OneDrive Fundamentals
OneDrive projects covering sync configuration, Files On-Demand, file availability states, and sync troubleshooting.

### SharePoint Management
SharePoint projects covering permission groups, permission inheritance, document libraries, metadata, and access management.
