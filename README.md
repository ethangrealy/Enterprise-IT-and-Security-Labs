# Enterprise Cloud and Security Labs

A practical portfolio of hands-on IT administration projects focused on Microsoft enterprise cloud environments.

This repository currently covers:

- Windows Server administration (e.g., Active Directory, Group Policy, DNS/DHCP, etc.)
- Microsoft 365 and Microsoft Entra administration

The projects are designed to demonstrate practical support, identity, access, and infrastructure administration skills through concise documentation and screenshots.

## Projects

### Windows Server

Hands-on Windows Server and Active Directory administration projects, including:

- User and group management
- Organisational Units
- Group Policy
- NTFS permissions and file shares
- Windows LAPS
- DNS and DHCP

[View Windows Server projects](./Windows-Server/)

### Microsoft 365 and Entra Tenant Management

Hands-on Microsoft 365 and Entra administration projects, including:

- User lifecycle management
- Password resets
- MFA and authentication methods
- Session revocation
- Licence assignment
- Group management
- Sign-in log analysis
- Authentication troubleshooting

[View Microsoft 365 and Entra projects](./M365-and-Entra/Entra-Tenant-Management/)

### Exchange Online

Brief experiences of Exchange Online use-cases and scenarios, including:

- Exchange Admin Center
- Inspecting user mailboxes
- Creating shared mailboxes
- Configuring mailbox delegation
- Full Access, Send As and Send on Behalf
- Creating distribution groups
- Managing distribution-group membership
- Aliases
- Message traces

[View Exchange Online projects](./M365-and-Entra/Exchange-Online/)

### SharePoint Management

Small demonstration of SharePoint activities and troubleshooting, including:

- SharePoint permission groups
- Configuring permission inheritance
- Restrictive access and ensuring actual permissions
- Libraries, folders and synced content

[View SharePoint Management projects](./M365-and-Entra/Sharepoint-Management/)

### OneDrive Fundamentals

Showcasing basic OneDrive functionalities and troubleshooting, including:

- OneDrive Sync Fundamentals
- Files On-Demand
- OneDrive Sync Troubleshooting
- Sync vs Permission vs Account Failure

[View OneDrive Fundamentals projects](./M365-and-Entra/OneDrive-Fundamentals/)

## Purpose

This repository documents practical lab work completed to build and demonstrate skills relevant to:

- IT support
- Help desk and service desk
- Microsoft 365 administration
- Identity and access management
- Systems administration
- Security operations

All environments use fictional users and lab data. Sensitive information is redacted before publication.
