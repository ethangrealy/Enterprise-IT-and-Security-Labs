# Windows Server Administration Projects

This section contains practical Windows Server and enterprise administration projects focused on identity, file services, workstation management, security, and core network services.

## Projects

### Active Directory
Built and managed an Active Directory environment including users, groups, organisational units, and administrative structure.

### Secure File Services
Configured shared folders, NTFS permissions, security groups, and controlled access to departmental resources.

### Enterprise Workstation Management
Managed Windows client devices in a domain environment, including domain joining, Group Policy, printer deployment, and drive mapping.

### Windows LAPS
Configured Windows LAPS to securely manage and rotate local administrator passwords for domain-joined devices.

### DHCP Server
Configured and administered DHCP services, including scopes, exclusions, reservations, and automatic IP address assignment.
